# PDFast File and Image PDF Generator

## Installation
### Clone Repository
```sh
git clone <repo-url>
cd <repo-folder>
```

### Install Dependencies
**Using Virtual Environment:**
```sh
python -m venv venv
# Activate Virtual Environment
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

pip install -r requirements.txt
```

**Or Directly:**
```sh
pip install -r requirements.txt
```

## Usage
```sh
python main.py
```
